# Task Tracker - Flask CRUD Application

Backend-driven task management web application built with Python Flask and SQL database implementing full CRUD functionality.

## Features
- Create, Read, Update and Delete tasks
- Modular application architecture
- RESTful routing
- Database integration with SQLAlchemy
- Service and repository pattern structure
- User interface with HTML/CSS templates

## Tech Stack
- Python
- Flask
- SQLAlchemy
- HTML/CSS
- Jinja Templates
